==================================================存储单元分配=================================================================

-Code：代码的大小
-RO：常量所占空间
-RW：程序中已经初始化的变量所占空间
-ZI：未初始化的static和全局变量以及堆栈所占的空间

-Flash占用大小 = Code+RO+RW     最大240k
-SRAM 占用大小 = RW+ZI          最大128k
